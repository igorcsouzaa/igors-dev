# Igor's dev - Site Comercial

Este é um site comercial moderno e responsivo, desenvolvido utilizando apenas **HTML, CSS e JavaScript puro**, conforme solicitado.

## Estrutura do Projeto

O projeto está organizado nos seguintes arquivos principais:

- `index.html`: Contém toda a estrutura semântica do site, incluindo todas as seções solicitadas (Início, Quem Somos, Serviços, Portfólio, Depoimentos/Parcerias e Contato).
- `style.css`: Responsável pela estilização completa do site, garantindo um design profissional, limpo e totalmente responsivo (Mobile First).
- `script.js`: Implementa as funcionalidades leves em JavaScript, como o menu mobile, rolagem suave e o carrossel de portfólio.
- `assets/`: Pasta contendo imagens de placeholder para o Hero, Portfólio e logos de parceiros.

## Funcionalidades Implementadas

- **Design Responsivo:** O layout se adapta perfeitamente a diferentes tamanhos de tela (desktop, tablet e mobile).
- **Menu Mobile:** O menu de navegação se transforma em um ícone "hambúrguer" em telas menores, controlado via JavaScript.
- **Rolagem Suave:** Navegação suave entre as seções do site.
- **Carrossel de Portfólio:** Galeria de projetos interativa com botões de navegação (Próximo/Anterior).
- **Estilo Moderno:** Uso de variáveis CSS, fontes modernas e um esquema de cores focado em profissionalismo e tecnologia.

## Como Visualizar

Para visualizar o site, basta abrir o arquivo `index.html` em qualquer navegador web moderno.

```bash
# 1. Descompacte o arquivo ZIP.
# 2. Abra o arquivo index.html no seu navegador.
```
